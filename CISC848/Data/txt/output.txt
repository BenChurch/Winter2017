## National Cyber Awareness System

### Vulnerability Summary for CVE-2017-2950

Original release date: 01/10/2017

Last revised: 01/17/2017

Source: US-CERT/NIST

#### Modified

This vulnerability has been modified since it was last analyzed by the NVD. It
is awaiting reanalysis which may result in further changes to the information
provided.

#### Overview

Adobe Acrobat Reader versions 15.020.20042 and earlier, 15.006.30244 and
earlier, 11.0.18 and earlier have an exploitable use after free vulnerability
in the XFA engine, related to layout functionality. Successful exploitation
could lead to arbitrary code execution.
